# ئۇيغۇرچە تەرجىمە قۇرالى — MdTranslationAiTool

كۆپ خىل AI تەرجىمان ئارقىلىق Markdown، TXT، HTML، PDF ھۆججەتلەرنى ئۇيغۇرچىگە (ۋە باشقا تىللارغا) تەرجىمە قىلىدىغان كۈچلۈك قۇرال.

---

## ئىقتىدارلار

- **7 خىل AI تەرجىمان:** Claude، Gemini، GPT/Codex، Groq، DeepSeek، Mistral، Cohere
- **4 خىل ھۆججەت فورماتى:** `.md` `.txt` `.html` `.pdf`
- **بۆلەك-بۆلەك تەرجىمە** — ئۇزۇن ھۆججەتلەر قىسقارمايدۇ، سېكشىن بويىچە ئايرىلىدۇ
- **تەلەپ.txt** — ھەر فولدېرگە ئالاھىدە تەرجىمە تەلىپى قوشسا بولىدۇ
- **ئاتالغۇ خەزىنىسى** — تېخنىكىلىق ئاتالغۇلارنى ئالدىن بەلگىلەش
- **تەرجىمە كەشى** — ئوخشاش بۆلەكنى قايتا يوللامايدۇ (API پۇل تېجەيدۇ)
- **داۋاملاشتۇرۇش** — توختاپ قالسا، تامام بولغان ھۆججەتلەرنى ئاتلايدۇ
- **ئالدىن كۆرۈش** — 1-بۆلەكنى سىناق تەرجىمە قىلىپ كۆرسىتىدۇ
- **Dry Run** — تەرجىمە قىلماي، بۆلەك-سان ۋە ھۆججەت سانىنى سانايدۇ
- **خاتىرە فايل** — ھەر سېانس `logs/` پاپكىسىغا ساقلىنىدۇ
- **9 خىل نىشان تىل** — ئۇيغۇرچە، تۈركچە، قازاقچە، ئۆزبېكچە...

## ئورناتىش

**تەلەپ:** Python 3.10 ۋە ئۇنىڭدىن يۇقىرى

```bash
pip install -r requirements.txt
```

## ئىجرا قىلىش

### Windows دا (ئەڭ ئاسان)
`باشلاش.bat` نى ئىككى قېتىم چېكىڭ — كۈتۈپخانىلارنى ئاپتوماتىك ئورنىتىپ قۇرالنى ئاچىدۇ.

### كونسولدا
```bash
python main.py
```

---

## ئىشلىتىش تەرتىپى

### 1. API كۇنۇپكىسىنى ساقلاش
`API كۇنۇپكا` بېتىگە كىرىپ ئىشلىتىدىغان AI نىڭ كۇنۇپكىسىنى كىرگۈزۈپ `ساقلا` نى بېسىڭ.

| AI | API كۇنۇپكا ئالىش ئۇلانمىسى |
|----|--------------------------|
| Claude | https://console.anthropic.com |
| Gemini | https://aistudio.google.com/app/apikey |
| GPT/Codex | https://platform.openai.com/api-keys |
| Groq | https://console.groq.com/keys |
| DeepSeek | https://platform.deepseek.com/api_keys |
| Mistral | https://console.mistral.ai/api-keys |
| Cohere | https://dashboard.cohere.com/api-keys |

### 2. تەرجىمە قىلىش
1. `تەرجىمە` بېتىدە AI ۋە مودېل تاللاڭ
2. `كۆزىتىش` تۈگمىسى ئارقىلىق فولدېر تاللاڭ
3. نىشان تىلنى تاللاڭ
4. `تەرجىمىنى باشلا` نى بېسىڭ

### 3. نەتىجە
تاللىغان فولدېرنىڭ ئاتىنىڭ ئارقىسىغا `-ug` قوشۇلۇپ يېڭى فولدېر قۇرۇلىدۇ:

```
countries/          ← ئەسلى فولدېر
    china.md
    japan.md
countries-ug/       ← تەرجىمە نەتىجىسى
    china.md
    japan.md
```

## تەلەپ.txt ئىشلىتىش

تەرجىمە قىلىنىدىغان فولدېرنىڭ ئىچىگە `تەلەپ.txt` ھۆججىتى قوشسىڭىز، قۇرال ئاپتوماتىك ئوقۇپ تەرجىمە تەلىپى سۈپىتىدە ئىشلىتىدۇ.

```
# تەلەپ.txt نۇمۇنەسى

- تەرجىمە ئۇيغۇر يازما تىلى ئاساسىدا بولسۇن
- database → سانلىق مەلۇمات ئامبىرى دەپ تەرجىمە قىلىڭ
- كودنى تەرجىمە قىلماڭ
```

> نۇمۇنە ئۈچۈن `تەلەپ_نۇمۇنە.txt` نى كۆرۈڭ.

---

## ئاتالغۇ خەزىنىسى

`ئاتالغۇ خەزىنىسى` بېتىدە تېخنىكىلىق ئاتالغۇلارنى قوشسىڭىز بولىدۇ. مەسىلەن:

| ئەسلى | تەرجىمە |
|------|--------|
| server | مۇلازىمېتىر |
| database | سانلىق مەلۇمات ئامبىرى |
| algorithm | ئالگورىزم |

بۇ ئاتالغۇلار ھەر تەرجىمىدە AI غا ئاپتوماتىك يوللىنىدۇ.

```
├── تەلەپ_نۇمۇنە.txt            # تەلەپ ھۆججىتى نۇمۇنىسى

├── translators/
│   ├── claude_translator.py
│   ├── gemini_translator.py
│   ├── openai_translator.py
│   ├── groq_translator.py
│   ├── deepseek_translator.py
│   ├── mistral_translator.py
│   └── cohere_translator.py
├── processors/

│   ├── txt_processor.py
│   ├── html_processor.py
│   └── pdf_processor.py
├── utils/
│   ├── cache.py                 # تەرجىمە كەشى
│   ├── glossary.py
│   ├── config.py                # API كۇنۇپكا ساقلاش

```

## ئەسكەرتىش

- رېفرېنس سېكشىنى (`References`, `Bibliography`, `مەنبەلەر`...) تەرجىمە قىلىنمايدۇ، ئەسلىسى ساقلىنىدۇ
- ئۇلانمىلار، رەسىملەر، كود بلوكلار ئەسلى ھالەتتە قالىدۇ
- API كۇنۇپكىلىرى `settings.json` غا شىفىرلىنىپ ساقلىنىدۇ
